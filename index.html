Jupyter Notebook Viewer for Linux

A utility for previewing Jupyter notebooks with a double click.

Jupyter Notebook Viewer (Linux)

  • This is a utility which lets you preview jupyter ntebooks faster on your Linux machine, with a double click.
  • Requires no internet and is faster.

Installation

Download the latest release from https://github.com/jithu7432/nb-viewer/releases/ and follow the instructions in the README.md for installation.

Usage

  • When using for the first time, right click a .ipynb file,

    • properties » open with»Notebook Viewer

  • From the next time, view Jupyter notebooks with a double click!
